This is a C++ program to price a European-style call or put option using each of the methods 
indicated in the code.

For the program output test, assume that the stock price process is geometric Brownian motion
with volatility sigma = 0:3, initial asset price S(0) = 100, constant risk-free interest rate r = 0.05, dividend yield d = 0.02, option type call, strike K = 110, and maturity T = 1 year.
